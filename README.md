# 拼图工具

一个免费的在线拼图工具，支持图片拼接、长图拼接、宫格拼图、横向拼接、截图拼接和满屏水印。

在线使用：[拼图工具 pintugongju.com](https://pintugongju.com)

## 功能

- 多张图片本地拼接
- 支持网格、纵向、横向三种布局
- 支持拖拽调整图片顺序
- 支持无缝拼接
- 支持满屏文字水印
- 支持下载 PNG 图片
- 图片在浏览器本地处理，不主动上传服务器

## 使用方式

直接打开 `index.html` 即可使用。

如果部署到静态托管平台，可以上传整个目录：

- `index.html`
- `styles.css`
- `app.js`

## 技术说明

这是一个纯前端静态网页，不需要后端服务。
